Equipe

Estoque de tecidos

Saldo em kg por tecido e por cor. As saídas são lançadas automaticamente quando você salva uma OS (consumo do enfesto). Use os botões para registrar entradas (compras) e ajustes manuais.

Estoque de corte

Peças cortadas, em peças por tecido e cor. As etapas são sobrepostas: o volume da OS fica sempre no campo da última etapa marcada. Aqui ficam as OSs cuja última etapa marcada é Corte; ao marcar outra etapa (Costura, Retirada de fios, Expedição…) o volume sai daqui e vai para o campo dela. Use os botões para contagem física e ajustes manuais.

Costurando

Peças em costura, em peças por tecido e cor. Recebe a OS quando a etapa Costura é a última marcada; sai quando outra etapa vira a última marcada. A coluna OS aqui mostra todas as OSs que já passaram pela Costura. Use os botões para contagem física e ajustes manuais.

Retirada de fios

Peças na retirada de fios, em peças por tecido e cor. Recebe a OS quando a etapa Retirada de fios é a última marcada; sai quando outra etapa vira a última marcada. Use os botões para contagem física e ajustes manuais.

Expedição

Peças em expedição, em peças por tecido e cor. Recebe a OS quando a etapa Expedição é a última marcada (o volume sai do campo anterior). Use os botões para contagem física e ajustes manuais.

Configurações

Exportar, importar e limpar dados
Pasta de backup automático (JSON)
A cada mudança nos cadastros ou OS, um arquivo gerador-os-dados.json com TODOS os dados é gravado na pasta conectada. Aponte para a pasta do programa pra ter o JSON junto do código (ideal para versionar via git). Funciona apenas no Chrome/Edge no desktop. Backup automático silencioso — não substitui o Supabase, é cópia espelho.

Pasta de PDFs gerados
Cada OS gerada salva o PDF nesta pasta automaticamente. Funciona apenas no Chrome ou Edge no desktop. Conecte uma vez — o navegador lembra entre sessões. Em outro computador, navegador ou no celular, é preciso conectar de novo (ou nada será salvo).
Carregando…
Backup e restauração
Além dos snapshots diários automáticos, você pode exportar/importar manualmente. "Limpar tudo" apaga TODOS os cadastros da equipe e é irreversível.
Popular com dados de exemplo
Preenche cadastros com dados baseados na ficha Dx7282 da referência (Moletom, cor Camel, coleção Inverno 2024, etc.) para você testar.
Operações em massa nos desenhos
Copia as etapas (e a ordem) configuradas no desenho de origem para os demais desenhos do mesmo modelo e variação. Ex.: Camiseta Básica só copia para outras Camisetas Básicas — não para Camiseta Bicolor nem Tricolor; Blusa Moletom Básica não copia para Blusa Moletom Tricolor. Apenas o campo de etapas é alterado — imagem, vínculos, componentes e demais campos ficam intactos.
Sincronização
Se você cadastrou itens em outra aba/sessão e eles não estão aparecendo aqui (ex.: etapas ou tarefas novas que não surgem nas opções dos desenhos), force uma releitura completa do servidor.
Snapshots automáticos (últimos 30 dias)
Uma cópia completa dos dados é gravada automaticamente uma vez por dia. Se algo der errado, você pode restaurar um dia anterior.
Clique em "Atualizar lista" para carregar.
Usuários e permissões
Somente admin pode gerenciar papéis. Usuário comum só pode criar/editar OS — não mexe em cadastros nem apaga dados.
